# Cloud Task Manager

A cloud-based task manager web app built with HTML, CSS, JavaScript and Firebase.

## Features
- Add, edit, delete tasks
- Mark tasks as complete
- Real-time cloud storage with Firebase Firestore
- Filter tasks by All / Active / Completed
- Deployed live on Firebase Hosting
